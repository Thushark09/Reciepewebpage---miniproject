# Recipe Website

## Project Description

The **Recipe Website** is a simple HTML-based webpage that displays a homemade recipe in a structured and user-friendly format. The webpage includes information such as preparation time, ingredients, cooking steps, and nutritional information.

This project was created to practice **HTML fundamentals and semantic HTML elements** while building a real-world example of a recipe webpage.

## Features

* Clear recipe description
* Image of the dish with caption
* Preparation and cooking time details
* List of ingredients
* Step-by-step cooking instructions
* Highlighted cooking tip
* Nutritional information section
* Semantic HTML structure using modern HTML5 tags

## Technologies Used

* HTML5

## HTML Concepts Practiced

* Semantic elements (`header`, `main`, `section`, `footer`)
* Lists (`ul`, `ol`, `li`)
* Description lists (`dl`, `dt`, `dd`)
* Image elements (`figure`, `figcaption`)
* Text formatting (`strong`, `mark`, `small`)
* Page structuring elements (`hr`, headings)

## Project Structure

The webpage is organized into the following sections:

* Header (Recipe title and introduction)
* Main Content

  * Recipe description
  * Preparation time
  * Ingredients
  * Cooking steps
  * Nutritional information
* Footer (Copyright information)

## Purpose of the Project

The purpose of this project is to:

* Learn HTML page structure
* Practice semantic HTML elements
* Understand how to organize content in a webpage
